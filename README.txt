Reduce Shoe Returns drastically by using our AI-powered tool that helps your customers finding the right shoe size!

== Description ==

mySHOEFITTER is an innovative web software designed to revolutionize the way you shop for footwear online. Through advanced technology, we allow customers to accurately **determine their perfect shoe size** from the comfort of their own home.

**Simply by taking a photo of their foot,** customers can bypass the guesswork and uncertainty that often comes with selecting shoe sizes online. Our software, once implemented into your online shop, analyzes the photo and calculates the precise dimensions of the foot. Using this data, it recommends the ideal shoe size for every style and brand you carry.

Beyond providing convenience to customers, mySHOEFITTER also offers significant benefits to retailers. By **reducing size-related returns and exchanges,** it improves customer satisfaction and helps businesses operate more efficiently.

== Installation ==

= Minimum Requirements =

* PHP 7.4 or greater is required (PHP 8.0 or greater is recommended)
* MySQL 5.6 or greater, OR MariaDB version 10.1 or greater, is required

= Automatic installation =
	
Automatic installation is the easiest option -- WordPress will handle the file transfer, and you won’t need to leave your web browser. To do an automatic install of mySHOEFITTER, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”

In the search field type myshoefitter,” then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the mySHOEFITTER plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
